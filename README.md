## Issues I Ran Into

**Finding the Extensions panel in Cursor**
Cursor opens in an agent/composer view by default, which doesn't show 
the standard editor interface. Had to find and click "Editor Window" 
in the top right to switch to the editor where extensions are accessible.

**Git not recognised in terminal**
Running the git clone command initially returned an error — git wasn't 
installed on my system. Downloaded and installed Git separately from 
git-scm.com, reopened the terminal, and reran the command successfully.
